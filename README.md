# 🎵 Music Streaming Web App

A modern full-stack MERN Music Streaming Web Application built with React.js, Node.js, Express.js, and MongoDB. The application provides a smooth and responsive user experience with features like authentication, playlist management, album and artist sections, and an interactive music player.

🚀 Live Demo: https://music-app-kappa-sand.vercel.app/

## ✨ Features
- User Authentication & Authorization
- Music Streaming Player
- Playlist & Album Management
- Responsive User Interface
- REST API Integration
- Secure JWT Authentication
- Full Stack MERN Architecture

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT Authentication
